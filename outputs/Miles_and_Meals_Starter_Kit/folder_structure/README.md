# Production Folder Structure

Use this folder tree for Miles & Meals media production.

## Folders

- `RAW/` - untouched original photos and videos.
- `Enhanced/` - AI-enhanced media before manual creative editing.
- `Lightroom_Ready/` - photo assets prepared for or exported from Lightroom.
- `Video_Ready/` - enhanced clips ready for CapCut.
- `CapCut_Projects/` - active video project files and supporting assets.
- `Instagram_Ready/` - final social-ready exports.
- `Published/` - assets that have been posted.
- `Archive/` - completed campaigns and historical projects.
- `Assets/` - logos, graphics, overlays, and brand media.
- `Music/` - approved music and audio notes.
- `Presets/` - Lightroom and CapCut preset files or specs.
- `Templates/` - reusable content, caption, and workflow templates.
- `Brand/` - brand guidelines, identity, media kit, and marketing assets.
- `Automation/` - n8n workflows, scripts, API notes, and dashboard plans.
