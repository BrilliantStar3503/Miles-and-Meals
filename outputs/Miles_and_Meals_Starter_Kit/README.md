# Miles & Meals Starter Kit

This starter kit turns the Miles & Meals master blueprint into a working content production workspace.

## Contents

- `00_Master_Blueprint.md` - central source of truth for the brand and ecosystem.
- `01_Operating_Manual.md` - repeatable production workflow from capture to archive.
- `02_Approval_Checklist.md` - review gate before anything is published.
- `03_Caption_Studio.md` - caption, hashtag, SEO, and ALT text templates.
- `04_Roadmap.md` - phased implementation plan.
- `folder_structure/` - empty production folders with `.gitkeep` placeholders.
- `templates/` - reusable metadata, post, analytics, and prompt templates.

## Guiding Principle

Every workflow, automation, feature, and decision must answer:

> Does this reduce repetitive work while preserving creative quality and authentic storytelling?

If yes, it belongs in the Miles & Meals ecosystem. If no, simplify it or discard it.
